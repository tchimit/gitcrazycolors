Git Demo Page!

Using git branches: We have our main branch called master. At the same time we open our second branch called for example crazycolors typing "git branch crazycolors" and start making changes to files. We still can return to our master branch typing "git checkout master" but changes to crazycolors won't appear on it.

Merging second branch to our master: on master branch add "git merge crazycolors"

P.S. key "q" quites git log list; if there's a merging conflict type "git merge --abort"
