Git Demo Page!

1. Using git branches: We have our main branch called master. At the same time we open our second branch called for example crazycolors typing "git branch crazycolors" and start making changes to files. We still can return to our master branch typing "git checkout master" but changes to crazycolors won't appear on it.

2. Merging second branch to our master: on master branch add "git merge crazycolors"

3. Pressing "q" quites git log list.

4. If there's a merging conflict type "git merge --abort"

5. To download your file to GitHub: open new repository on github and name it. Copy and paste "git remote add origin https://github.com/tchimit/gitcrazycolors.git" then type "git push -u origin master"

6. If your collaborator adds new commit to your file on github (you need to give permission first) you won't see it on your machine (only on github), therefore you need to download the file back to your computer: "git pull origin master"

7. To download repository from github copy the url address and type "git clone https://..." do not clone it to your existing repository. make new folder for it.
